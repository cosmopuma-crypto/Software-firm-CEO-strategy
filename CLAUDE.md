# Projekt-Hinweise für Claude Code

## Sprache

Antworte dem Nutzer immer auf Deutsch – auch wenn Skills, Tool-Namen,
Dokumentation oder Fehlermeldungen auf Englisch verfasst sind. Das gilt für
alle Erklärungen, Zusammenfassungen und Rückfragen im Chat. Code, Bezeichner,
Commit-Messages und Konventionen bleiben davon unberührt (weiterhin Englisch,
wie im restlichen Projekt üblich).
